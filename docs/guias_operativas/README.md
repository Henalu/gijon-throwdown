# Guias operativas por perfil

Este pack esta pensado para enviar a personas reales del evento sin tener que mandarles una biblia de producto.

Cada archivo responde a la pregunta:

- que panel uso
- que puedo hacer
- que no me toca
- y que rutina deberia seguir para no perderme

## Que guia enviar a cada persona

- `guia_superadmin.md`: para quien gestiona accesos, roles y el control total de la plataforma
- `guia_admin.md`: para quien lleva la operativa general del evento
- `guia_admin_validador.md`: para quien revisa y valida resultados oficiales
- `guia_voluntario.md`: para quien mete live en pista
- `guia_juez.md`: para quien arbitra dentro del flujo de voluntariado
- `guia_atleta.md`: para quien entra como atleta a consultar su cuenta y seguir el evento

## Nota importante para no liar perfiles

- `Juez` no es un rol global separado: es un perfil visible dentro de `volunteer`
- `Admin validador` no es un rol nuevo: es un `admin` con capacidad de validar scores
- `Superadmin` hereda todo lo de admin y ademas controla usuarios y permisos

## Frase corta para recordar

> No todo el mundo entra a la misma web para hacer lo mismo, y eso aqui no es un defecto: es precisamente lo que evita que el evento acabe funcionando con fe, cafe y mensajes cruzados.

## Nota operativa nueva sobre live

- el live ya permite guardar parciales opcionales en cualquier WOD
- cuando una calle se cierra, se puede dejar observacion para validacion
- si el heat alcanza el cap, la edicion live normal se bloquea y el sistema cierra lo que siga abierto
